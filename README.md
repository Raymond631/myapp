# myapp

## 简介

用于快速搭建基于SSM框架的Web后端项目

## 基本内容

* 统一响应
* 统一异常处理
* 统一日志
* 统一鉴权
* 密码加密
* 跨域
* 工具类
* 基础接口：验证码、登录、注册、注销登录

## 使用框架

* Spring Boot
* Spring MVC
* Mybatis
* [Sa-Token](https://github.com/dromara/sa-token)

> 部分代码参考了[RuoYi-Vue](https://gitee.com/y_project/RuoYi-Vue)框架

## 数据存储

* MySQL
* Redis

## 其他依赖

* Druid（连接池）
* Lombok（代码生成）
* Validation（参数校验）
* Kaptcha（验证码）
* Argon2（加密算法）
